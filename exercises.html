Exercise 1

Array of Salaries

Work with an int salaries[100] array and build helper functions to:

  • Read n salaries (reject sizes > 100).
  • Print salaries on one line separated by “-”.
  • Compute the average salary and count how many are ≥ average.
  • Reverse the array in-place.
  • Shift the array right by one position.

Sample interaction:

Enter n: 110
Invalid Size! Try again
Enter n: 5
Enter 5 salaries: 4000 3200 5000 2600 1500
Array of salaries is [4000 – 3200 – 5000 – 2600 – 1500]
Average of Salaries is 3260
Count of Employees having a salary greater than or equal to 3260 is 2
The new array after calling Reverse is [1500 – 2600 – 5000 – 3200 – 4000]
The new array after calling Shift Right is [4000 – 1500 – 2600 – 5000 – 3200]
Exercise 2

Random Arrays of Positive and Negative Values

Generate n random integers between min and max inclusive, then copy strictly positive values to APOS and strictly negative values to ANEG.

Tips:

  • Seed the RNG with srand(time(NULL)).
  • Use r = min + rand() % (max - min + 1); for each element.
  • Display both arrays for verification (ignore zeros).

Sample interaction:

Enter n: 110
Invalid Size! Try again
Enter n: 10
Enter min and max: -10 10
Array of random values between -10 and 10 is -9 3 5 10 1 4 -4 0 3 4
Array of strictly positive values is 3 5 10 1 4 3 4
Array of strictly negative values is -9 -4
